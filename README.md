Brandy-themed storefront with cart, categories, search, sorting, and warm amber/wood palette.

## Quick start
1. Unzip this folder.
2. In the folder, run:
   ```bash
   npm install
   npm run dev
   ```
3. Open the local URL it prints.

## Build for deploy
```bash
npm run build
```
Deploy the generated `dist/` folder to Netlify, Vercel, or Cloudflare Pages.

## Customize
- Update products / images in `src/App.jsx` (see `PRODUCTS`).
- Palette in `tailwind.config.js` (amber/wood/gold/cream).
